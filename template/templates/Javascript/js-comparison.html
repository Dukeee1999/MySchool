Comparisons

These are used to test an expression to be True or False.
Here are the comparison operators:

  • == : equal to
  • === : equal type and value
  • != : not equal
  • !== : not equal value or not equal type
  • > : greater than
  • >= : greater than or equal to
  • < : less than
  • <= : less than or equal to

Logical Operators

Logical operators handle the logic between variables/ values.

  • && : and
  • || : or
  • ! : not

Conditional

This operator assigns a value to a variable dependant on some condition.
Syntax: someVariable = (condition) ? value 1: value 2 


    var speeding = (speed <=60) ? "No":"Yes";

if the condition (speed <= 60) is true, then value 1 will be stored ("No"), else value 2 will be stored ("Yes").
