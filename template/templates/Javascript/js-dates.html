Dates

JavaScript Date Object allows dates to be used and can be initialized in a variety of ways.
JavaScript by default will display a date as a full text string.
There are 4 ways to initialize a DateObject. 

var express = require('express')
    new Date();
    new Date(year, month, day, hours, minutes, seconds, milliseconds);
    new Date(milliseconds);
    new Date(date string);
  • new Date() creates a new date Object with the current time and date.
  • new Date(year, month, day, hours, minutes, seconds, milliseconds) creates a new date Object with the specified time and date. (one and 2 digit years will be interpreted as the previous century)
  • new Date(milliseconds) creates a new date Object with the time and date as number of milliseconds from 1 January 1970 00:00:00 UTC. This is referred to as zero time.
  • new Date(date string) creates a date from a date string.
