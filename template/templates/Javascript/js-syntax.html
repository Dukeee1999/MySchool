Syntax and Variables

Syntax is a set of rules that defines the correctly structured statements/ expressions in a language, e.g. JavaScript requires you to declare, assign and compute variables in that order. JavaScript has two kinds of variables:

  • Fixed Values - Literals
  • Variable Values - Variables

Literals

String literals are text and must simply be written within single or double quotes.
Number literals are written with/without decimals.

Variables

Variables are used to store values that are subject to vary in the future.
JavaScript uses var as the keyword to declare one or more variables. They are then assigned with an equals sign.
A declared variable will start with an undefined value. Redeclaring a variable will not lose an already stored value.
However, these variables must follow a syntax:

  • the First character must be a letter, or an underscore (_), or a dollar sign ($). (Numbers are not allowed)
  • Other characters may be letters, digits, underscores, or dollar signs. 

    var x, y;
    x = 5;
    y = 6;

Note

JavaScript is case sensitive.
JavaScript programmers tend to use lower camel case. i.e. javaScript, camelCase, etc.
