Introducing JavaScript

JavaScript is a text-based programming language that allows for the ability to make web pages interactive. Whereas HTML and CSS are languages that allow for style and structure within a web page, JavaScript allows for web pages to have interactive elements that engage the user. JavaScript allows you to:

  • Create dynamically updating content.
  • Control multimedia and animate images.
  • Run code in response to certain events occurring on a web page.
  • Make a website a lot more interactive and engaging for the user.

JavaScript is run in order, from top to bottom. This means that the programmer must be careful of the order of things such that something is not referenced before it is declared, e.g. using a function before it is made. This would return an error and crash the code. This makes JavaScript an interpreted language. JavaScript is also a dynamic language meaning it has the ability to update the display of a web page to show different things reacting to different events. A web page without this ability is referred to as static and shows the same content constantly.
