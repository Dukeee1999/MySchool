Introducing Frameworks

A web application framework is a piece of software that allows the programmer to create and run web applications.
A framework makes web development quicker and more convenient through providing basic patterns for building reliable applications. This way they reduce the need for overtly repetitive code.e.g. rewriting some code block each time a new website is created.
Web application frameworks also allow the programmer to create unique features, being highly customizable.
Framework functionality involves fundamental operations needed to run web applications:

  • URL routing
  • Input form managing and validation
  • Database connections and data manipulation
  • And more...

Bottle

Bottle is a lightweight micro web-framework for the Python language.
It is perfect for small web applications and rapid prototyping.
